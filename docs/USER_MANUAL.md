## 1. 시작하기

### 설치 및 실행
```bash
git clone https://github.com/JonghwaCho/english-master.git
cd english-master
pip3 install -r requirements.txt
python3 server.py
```
브라우저에서 `http://127.0.0.1:5294` 접속

### 필요 환경
- Python 3.8+
- Flask, youtube-transcript-api, gunicorn
- 모던 브라우저 (Chrome, Safari, Firefox)

### 로그인 / 회원가입
접속하면 로그인 화면이 먼저 나타납니다.
1. **회원가입** 탭 → 이메일 + 비밀번호(8자 이상) 입력
2. **이메일 인증**: 가입하면 인증 메일이 발송됩니다. 메일함의 **인증 링크를 클릭하면 인증 완료 + 자동 로그인**됩니다. 인증 전에는 로그인할 수 없습니다.
   - 메일이 안 왔으면 로그인 화면의 **"인증 메일 재발송"** 버튼을 누르세요.
   - (참고) **구글 로그인** 계정과 **서비스의 첫 운영자 계정**은 인증 없이 바로 사용할 수 있습니다.
3. 이후에는 **로그인** 탭에서 같은 이메일/비밀번호로 접속
4. 로그아웃: 좌측 사이드바 하단의 **로그아웃** 버튼
5. **구글 로그인**: 서버에 구글 OAuth가 설정된 경우 "Google로 계속하기"로 바로 로그인 가능
6. **AI 키**: 설정 화면에서 개인 API 키를 넣으면 내 키로, 비워두면 서버 기본 키로 AI 기능(직독직해·단어뜻 등)이 동작합니다

### 비밀번호를 잊었을 때
로그인 화면의 **"비밀번호를 잊으셨나요?"**를 누르고 가입한 이메일을 입력하면 **재설정 링크**가 메일로 발송됩니다(1시간 내 유효). 링크를 열어 새 비밀번호를 설정하면 바로 로그인됩니다.
- 구글 로그인 계정은 비밀번호가 없으므로 구글로 로그인하세요.
- (운영자) 재설정 메일도 인증 메일과 같은 `/admin` SMTP 설정을 사용합니다.

### 처음 시작할 때 (온보딩)
가입 후 첫 화면에 **환영 안내**와 **시작 체크리스트**가 나타납니다.
- **① 콘텐츠 추가 → ② 첫 학습 → ③ 첫 복습** 3단계를 안내대로 따라가면 됩니다.
- 대시보드 상단의 체크리스트는 각 단계를 완료할 때마다 자동으로 체크되고, **3단계를 모두 마치면 사라집니다**. (오른쪽 ✕로 미리 닫을 수도 있어요.)

> **운영자 안내**: 인증 메일을 실제로 발송하려면 `/admin` → **이메일(SMTP) 발송 설정**에서 발송에 쓸 메일 계정(SMTP 호스트/계정/비밀번호)을 입력하고 "테스트 메일 발송"으로 확인하세요. 설정 전에는 인증 메일이 나가지 않습니다.

> 클라우드 버전은 https://english-master.fly.dev 에서 접속할 수 있습니다.

| 📚 등록 콘텐츠 | 추가한 영상/텍스트 수 |

- **문장 복습 / 단어 복습**: 복습 대기 수와 함께 바로 이동

## 3. 콘텐츠 관리

### 콘텐츠 추가 방법

#### YouTube 영상
1. YouTube URL 입력
2. 자동으로 자막(transcript) 추출
3. 자막이 없으면 가사 검색 또는 수동 입력 가능

#### 텍스트 직접 입력
1. "텍스트" 탭 선택
2. 영어 텍스트 붙여넣기
3. 제목 입력 후 저장

#### URL에서 가져오기
1. "URL" 탭 선택
2. 웹페이지 URL 입력
3. 미리보기 후 저장 (기사, 블로그 등)

#### 파일 업로드
- 지원 형식: .txt, .pdf, .html
- 최대 16MB

### 카테고리 관리
- 좌측 사이드바 "카테고리" 영역에서 `+` 버튼으로 추가
- 콘텐츠를 카테고리별로 분류
- 카테고리별 필터링 가능

### 플레이리스트
- YouTube 플레이리스트 URL 등록
- 자동 동기화 (30분 간격)
- 활성화/비활성화 토글

### 요금제와 콘텐츠 등록 한도

콘텐츠 등록 수는 요금제에 따라 제한됩니다. 좌측 사이드바 하단의 **요금제 배지**에서 현재 사용량(예: `2/5`)과 남은 개수를 확인할 수 있습니다.

| 요금제 | 가격 | 콘텐츠 한도 | 주기 |
|--------|------|-------------|------|
| 무료(free) | 0원 | 1개 | 평생 |
| 베이직(basic) | 월 3,000원 | 5개 | 매월 |
| 프리미엄(premium) | 월 5,000원 | 10개 | 매월 |
| 맥스(max) | 월 7,000원 | 20개 | 매월 |

- **카운트 시점**: 콘텐츠를 받아와 **전체학습/개별학습을 할 수 있는 상태로 등록되는 순간** 1개가 차감됩니다.
- **삭제해도 복구되지 않음**: 등록한 콘텐츠를 지워도 이번 주기 사용량은 되돌아오지 않습니다.
- **월 리셋**: 유료 요금제는 **구독을 시작한 날짜 기준**으로 매월 한도가 초기화됩니다. 무료는 평생 누적입니다.
- **한도 초과 시**: 등록이 막히고 "요금제를 업그레이드하세요" 안내가 표시됩니다.
- 가격과 한도는 관리자가 조정할 수 있으며, 요금제 변경은 현재 관리자가 처리합니다(자동 결제는 추후 지원 예정).

## 4. 학습하기

### 4.1 문장별 학습

영상/텍스트를 선택하면 문장 단위로 학습합니다.

#### 점진적 단어 노출 시스템
처음 문장이 표시될 때 모든 단어가 `·····`으로 마스킹됩니다.

| 다시듣기 횟수 | 노출 비율 |
|-------------|----------|
| 처음 | 0% (모든 단어 숨김) |
| 1회 | 20% |
| 2회 | 50% |
| 3회 | 75% |
| 4회 | 100% (전체 공개) |

#### 카라오케 하이라이트
- YouTube 영상의 타이밍에 맞춰 현재 재생 중인 단어가 파란색으로 하이라이트
- 설정에서 "카라오케 힌트 시작 회차"를 조절하면, 마스킹된 단어도 하이라이트 시 잠깐 보였다가 다시 숨겨짐

#### 버튼
- **▶ 다시 듣기**: 문장 다시 재생 + 단어 추가 노출
- **모르겠어요**: 모르는 문장으로 저장 → 에빙하우스 레벨 0
- **알아요!**: 아는 문장으로 저장 → 에빙하우스 레벨 1

#### AI 도구
- 📋 직독직해: 단어/청크/전체/팁 형식으로 번역
- 🔄 유사문장: 비슷한 패턴의 문장 생성
- 📖 문법설명: 문장 구조 분석
- 🔤 단어설명: 각 단어의 뜻/품사/발음/예문

### 4.2 전체학습 (일반 영상)

영상 전체를 연속 재생하며 문장을 따라갑니다.

#### 컨트롤 바 (상단 고정)
| 버튼 | 기능 |
|------|------|
| 일시정지 | 영상 일시정지/재개 |
| 해석 ON/OFF | 한국어 번역 표시 토글 |
| 단어 ON/OFF | Word Flash 카드 표시 토글 |
| 이모지 OFF | 가사의 이모지/음표 기호 필터링 |
| 종료 | 전체학습 종료 |

#### 단축키
- `R` = 모르는 문장으로 표시
- `Y` = 아는 문장으로 표시
- `Space` = 일시정지

#### Word Flash
- 현재 재생 중인 문장의 단어가 카드 형태로 순차 표시
- 카드 클릭 시 해당 단어 + 앞뒤 단어를 "모르는 단어"에 저장

#### 모르는 문장 클릭 동작
- 문장 클릭 → 현재 문장 끝까지 재생
- 재생 완료 후 → 모르는 문장으로 스크롤 + 설정 속도로 재생
- 재생 완료 → 다음 문장부터 자동 재개

### 4.3 전체학습 (임베딩 차단 영상 / TTS)

YouTube 임베딩이 차단된 영상은 TTS(음성합성)로 학습합니다.

#### 옵션
- **YouTube에서 열기**: 별도 팝업으로 YouTube 재생
- **TTS로 가사 읽기**: TTS 순차 재생 모드

#### TTS 컨트롤 바
| 버튼 | 기능 |
|------|------|
| 일시정지 | TTS 일시정지/재개 |
| 해석 ON/OFF | 한국어 번역 표시 |
| 이모지 OFF | 이모지 필터링 |
| 종료 | TTS 학습 종료 |

#### 모르는 문장 클릭 동작
- 현재 TTS 재생 끝까지 유지
- 완료 후 모르는 문장을 설정 속도로 재생
- 재생 완료 → 다음 문장부터 자동 재개

### 4.4 전체학습 리셋 기능 (시각적 리셋)
- 설정에서 "전체학습 시 모르는 문장 리셋" ON/OFF (기본: ON)
- ON: 전체학습 시작 시 이전에 모르는 문장으로 표시된 빨간 테두리만 시각적으로 초기화
- **데이터는 보존됨**: 모르는 문장 페이지에서는 그대로 모든 모르는 문장이 유지됨
- 매번 새로운 마음으로 학습하면서도 학습 이력은 누적됨

### 4.5 모르는 문장 누적 횟수
- 같은 문장을 여러 번 모르겠다고 표시하면 `unknown_count`가 증가
- 토스트 메시지: "모르는 문장으로 저장됨 (3회)"
- 모르는 문장 페이지에서 횟수 배지로 표시되며, 자주 틀린 문장이 상단에 정렬됨

## 5. 문장 복습

에빙하우스 망각곡선 기반 간격 반복 학습 시스템입니다.

### 복습 모드 선택
- **개별학습**: 한 문장씩 카드 형태로 복습
- **전체학습**: 전체 목록에서 한번에 체크

### 5.1 개별학습

#### 시작 화면
"시작하기" 버튼 → 클릭 후 학습 시작

#### 4버튼 리스닝 모드 (YouTube 문장)
듣기 먼저 모드가 켜진 경우, 문장을 숨기고 음성만 들려줍니다.

| 버튼 | 기능 |
|------|------|
| 🔊 다시듣기 | 영상/음성 다시 재생 |
| 👂 잘안들림 | 힌트 단어 표시 (문장 단어 + 랜덤 단어 섞어서 표시) |
| 📖 내용확인 | 문장 공개 → 잊어버렸어요/기억나요 선택 |
| ✅ 완벽숙지 | 팝업 없이 바로 다음 문장으로 (정답 처리) |

#### 잘안들림 힌트 기능
- 문장의 **모든 단어** + 동일 수의 **랜덤 영단어**가 희미하게 표시
- 단어 클릭 시:
  - 문장 단어 → ⭕ 파란색 (정답)
  - 랜덤 단어 → ❌ 빨간색 (오답)
- 리스닝 능력 향상에 도움

#### 내용 확인 후
- **잊어버렸어요** → 에빙하우스 레벨 0으로 리셋
- **기억나요!** → 다음 레벨로 진급
- 팝업에서 모르는 단어 선택 가능 (문장 내 단어 클릭)

#### 컨트롤
- 🔊 음성 ON/OFF: 자동 음성 재생 토글
- 📖 팝업 ON/OFF: 기억나요 클릭 시 팝업 표시 여부
- 🚪 학습 종료

### 5.2 전체학습
- 모든 복습 항목을 목록으로 표시
- 각 항목마다 "✅ 기억" / "❌ 잊음" 버튼
- 잊음 클릭 시 팝업에서 상세 확인 가능

## 7. 모르는 문장/단어

### 모르는 문장
- 학습 중 "모르겠어요"로 표시한 문장 목록
- 재생, 번역 확인, 삭제 가능
- 복습 레벨 표시
- **횟수 배지**: 각 문장 앞에 `❌ N회` 배지 표시 (1-2회: 연한 빨강, 3-4회: 중간, 5회+: 진한 빨강)
- **자동 정렬**: 자주 틀린 문장(unknown_count 높음)이 상단에 표시

### 모르는 단어
- 학습 중 선택한 모르는 단어 목록
- AI 뜻 자동 조회 (배치)
- 아는 단어로 변경 / 삭제

## 8. 통계

### 통계 카드 (8개)
글래스모피즘 디자인으로 각 항목별 아이콘 + 컬러 악센트

| 카드 | 아이콘 | 색상 |
|------|--------|------|
| 등록 콘텐츠 | 📚 | 파란색 |
| 전체 문장 | 💬 | 기본 |
| 아는 문장 | ✅ | 초록색 |
| 모르는 문장 | ❌ | 빨간색 |
| 완전 습득 | 👑 | 보라색 |
| 복습 대기 | ⏰ | 주황색 |
| 등록 단어 | 📝 | 기본 |
| 아는 단어 | 🎯 | 초록색 |

### 차트 (4개)
1. **문장 숙달** (도넛): 습득/안다/학습/모름/미학습 비율
2. **단어 숙달** (도넛): 습득/안다/모름/자주틀림 비율
3. **일별 학습** (라인): 최근 30일 문장/단어 학습 추이 (모든 날짜 표시)
4. **SRS 레벨** (막대): 레벨별 문장/단어 분포

### 필터링
- 콘텐츠별 필터 드롭다운으로 특정 영상의 통계만 확인 가능
- 차트 크기 슬라이더로 도넛 차트 크기 조절

### 전체학습 리셋
- **리셋 ON (기본)**: 전체학습 시작 시 이전 모르는 문장 초기화
- 리셋 OFF: 이전 상태 유지

- 원래 속도 (1x)
- 느리게 (0.5x)

- **듣기 먼저 (Listening)**: 문장 숨기고 음성 먼저 → 4버튼 UI
- 보기 먼저 (Reading): 문장 바로 표시

- 연결 테스트
