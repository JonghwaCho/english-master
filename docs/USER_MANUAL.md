## 1. 시작하기

### 설치 및 실행
```bash
git clone https://github.com/JonghwaCho/english-master.git
cd english-master
pip3 install -r requirements.txt
python3 server.py
```
브라우저에서 `http://127.0.0.1:5294` 접속

### 필요 환경
- Python 3.8+
- Flask, youtube-transcript-api
- 모던 브라우저 (Chrome, Safari, Firefox)

## 2. 대시보드

앱 실행 시 첫 화면입니다.

### 통계 카드 (6개)
| 카드 | 설명 |
|------|------|
| 📚 등록 콘텐츠 | 추가한 영상/텍스트 수 |
| 💬 전체 문장 | 등록된 총 문장 수 |
| ✅ 아는 문장 | known + mastered 문장 수 |
| ❌ 모르는 문장 | unknown 상태 문장 수 |
| ⏰ 복습 대기 | 에빙하우스 스케줄에 따라 복습 필요한 항목 수 |
| 🆕 미학습 | 아직 학습하지 않은 문장 수 |

### 빠른 시작
- **콘텐츠 추가**: 콘텐츠 페이지로 이동
- **학습 시작**: 학습하기 페이지로 이동
- **문장 복습 / 단어 복습**: 복습 대기 수와 함께 바로 이동

### 학습 진행률
- 학습 완료율 (%) 프로그레스 바
- 완벽히 아는 문장 (%) 프로그레스 바

## 3. 콘텐츠 관리

### 콘텐츠 추가 방법

#### YouTube 영상
1. YouTube URL 입력
2. 자동으로 자막(transcript) 추출
3. 자막이 없으면 가사 검색 또는 수동 입력 가능

#### 텍스트 직접 입력
1. "텍스트" 탭 선택
2. 영어 텍스트 붙여넣기
3. 제목 입력 후 저장

#### URL에서 가져오기
1. "URL" 탭 선택
2. 웹페이지 URL 입력
3. 미리보기 후 저장 (기사, 블로그 등)

#### 파일 업로드
- 지원 형식: .txt, .pdf, .html
- 최대 16MB

### 카테고리 관리
- 좌측 사이드바 "카테고리" 영역에서 `+` 버튼으로 추가
- 콘텐츠를 카테고리별로 분류
- 카테고리별 필터링 가능

### 플레이리스트
- YouTube 플레이리스트 URL 등록
- 자동 동기화 (30분 간격)
- 활성화/비활성화 토글

## 4. 학습하기

### 4.1 문장별 학습

영상/텍스트를 선택하면 문장 단위로 학습합니다.

#### 점진적 단어 노출 시스템
처음 문장이 표시될 때 모든 단어가 `·····`으로 마스킹됩니다.

| 다시듣기 횟수 | 노출 비율 |
|-------------|----------|
| 처음 | 0% (모든 단어 숨김) |
| 1회 | 20% |
| 2회 | 50% |
| 3회 | 75% |
| 4회 | 100% (전체 공개) |

#### 카라오케 하이라이트
- YouTube 영상의 타이밍에 맞춰 현재 재생 중인 단어가 파란색으로 하이라이트
- 설정에서 "카라오케 힌트 시작 회차"를 조절하면, 마스킹된 단어도 하이라이트 시 잠깐 보였다가 다시 숨겨짐

#### 버튼
- **▶ 다시 듣기**: 문장 다시 재생 + 단어 추가 노출
- **모르겠어요**: 모르는 문장으로 저장 → 에빙하우스 레벨 0
- **알아요!**: 아는 문장으로 저장 → 에빙하우스 레벨 1

#### AI 도구
- 📋 직독직해: 단어/청크/전체/팁 형식으로 번역
- 🔄 유사문장: 비슷한 패턴의 문장 생성
- 📖 문법설명: 문장 구조 분석
- 🔤 단어설명: 각 단어의 뜻/품사/발음/예문

### 4.2 전체학습 (일반 영상)

영상 전체를 연속 재생하며 문장을 따라갑니다.

#### 컨트롤 바 (상단 고정)
| 버튼 | 기능 |
|------|------|
| 일시정지 | 영상 일시정지/재개 |
| 해석 ON/OFF | 한국어 번역 표시 토글 |
| 단어 ON/OFF | Word Flash 카드 표시 토글 |
| 이모지 OFF | 가사의 이모지/음표 기호 필터링 |
| 종료 | 전체학습 종료 |

#### 단축키
- `R` = 모르는 문장으로 표시
- `Y` = 아는 문장으로 표시
- `Space` = 일시정지

#### Word Flash
- 현재 재생 중인 문장의 단어가 카드 형태로 순차 표시
- 카드 클릭 시 해당 단어 + 앞뒤 단어를 "모르는 단어"에 저장

#### 모르는 문장 클릭 동작
- 문장 클릭 → 현재 문장 끝까지 재생
- 재생 완료 후 → 모르는 문장으로 스크롤 + 설정 속도로 재생
- 재생 완료 → 다음 문장부터 자동 재개

### 4.3 전체학습 (임베딩 차단 영상 / TTS)

YouTube 임베딩이 차단된 영상은 TTS(음성합성)로 학습합니다.

#### 옵션
- **YouTube에서 열기**: 별도 팝업으로 YouTube 재생
- **TTS로 가사 읽기**: TTS 순차 재생 모드

#### TTS 컨트롤 바
| 버튼 | 기능 |
|------|------|
| 일시정지 | TTS 일시정지/재개 |
| 해석 ON/OFF | 한국어 번역 표시 |
| 이모지 OFF | 이모지 필터링 |
| 종료 | TTS 학습 종료 |

#### 모르는 문장 클릭 동작
- 현재 TTS 재생 끝까지 유지
- 완료 후 모르는 문장을 설정 속도로 재생
- 재생 완료 → 다음 문장부터 자동 재개

### 4.4 전체학습 리셋 기능
- 설정에서 "전체학습 시 모르는 문장 리셋" ON/OFF (기본: ON)
- ON: 전체학습 시작 시 이전에 모르는 문장으로 표시된 문장들을 자동 초기화
- 매번 새로운 마음으로 학습 가능

## 5. 문장 복습

에빙하우스 망각곡선 기반 간격 반복 학습 시스템입니다.

### 복습 모드 선택
- **개별학습**: 한 문장씩 카드 형태로 복습
- **전체학습**: 전체 목록에서 한번에 체크

### 5.1 개별학습

#### 시작 화면
"시작하기" 버튼 → 클릭 후 학습 시작

#### 4버튼 리스닝 모드 (YouTube 문장)
듣기 먼저 모드가 켜진 경우, 문장을 숨기고 음성만 들려줍니다.

| 버튼 | 기능 |
|------|------|
| 🔊 다시듣기 | 영상/음성 다시 재생 |
| 👂 잘안들림 | 힌트 단어 표시 (문장 단어 + 랜덤 단어 섞어서 표시) |
| 📖 내용확인 | 문장 공개 → 잊어버렸어요/기억나요 선택 |
| ✅ 완벽숙지 | 팝업 없이 바로 다음 문장으로 (정답 처리) |

#### 잘안들림 힌트 기능
- 문장의 **모든 단어** + 동일 수의 **랜덤 영단어**가 희미하게 표시
- 단어 클릭 시:
  - 문장 단어 → ⭕ 파란색 (정답)
  - 랜덤 단어 → ❌ 빨간색 (오답)
- 리스닝 능력 향상에 도움

#### 내용 확인 후
- **잊어버렸어요** → 에빙하우스 레벨 0으로 리셋
- **기억나요!** → 다음 레벨로 진급
- 팝업에서 모르는 단어 선택 가능 (문장 내 단어 클릭)

#### 컨트롤
- 🔊 음성 ON/OFF: 자동 음성 재생 토글
- 📖 팝업 ON/OFF: 기억나요 클릭 시 팝업 표시 여부
- 🚪 학습 종료

### 5.2 전체학습
- 모든 복습 항목을 목록으로 표시
- 각 항목마다 "✅ 기억" / "❌ 잊음" 버튼
- 잊음 클릭 시 팝업에서 상세 확인 가능

## 8. 통계

### 통계 카드 (8개)
글래스모피즘 디자인으로 각 항목별 아이콘 + 컬러 악센트

| 카드 | 아이콘 | 색상 |
|------|--------|------|
| 등록 콘텐츠 | 📚 | 파란색 |
| 전체 문장 | 💬 | 기본 |
| 아는 문장 | ✅ | 초록색 |
| 모르는 문장 | ❌ | 빨간색 |
| 완전 습득 | 👑 | 보라색 |
| 복습 대기 | ⏰ | 주황색 |
| 등록 단어 | 📝 | 기본 |
| 아는 단어 | 🎯 | 초록색 |

### 차트 (4개)
1. **문장 숙달** (도넛): 습득/안다/학습/모름/미학습 비율
2. **단어 숙달** (도넛): 습득/안다/모름/자주틀림 비율
3. **일별 학습** (라인): 최근 30일 문장/단어 학습 추이 (모든 날짜 표시)
4. **SRS 레벨** (막대): 레벨별 문장/단어 분포

### 필터링
- 콘텐츠별 필터 드롭다운으로 특정 영상의 통계만 확인 가능
- 차트 크기 슬라이더로 도넛 차트 크기 조절

## 9. 설정

### 학습 설정
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 영상 ON/OFF | YouTube 영상 표시 여부 | ON |
| 해석 ON/OFF | 한국어 번역 표시 | OFF |
| 음성 ON/OFF | TTS 음성 재생 | ON |
| TTS 음성 선택 | 영어 음성 종류 선택 | Google US English |

### 카라오케 힌트 시작 회차
다시듣기 시 카라오케 하이라이트로 마스킹된 단어가 잠깐 보이기 시작하는 회차
- 1회부터 / 2회부터 / **3회부터 (기본)** / 4회부터

### 전체학습 리셋
- **리셋 ON (기본)**: 전체학습 시작 시 이전 모르는 문장 초기화
- 리셋 OFF: 이전 상태 유지

### 모르는 문장 다시 재생 속도
- 원래 속도 (1x)
- 느리게 (0.5x)

### YouTube 문장 복습 모드
- **듣기 먼저 (Listening)**: 문장 숨기고 음성 먼저 → 4버튼 UI
- 보기 먼저 (Reading): 문장 바로 표시

### AI 설정
- 제공자: Gemini / Claude / ChatGPT
- API 키 입력
- 연결 테스트

### 단축키 설정
- 문장별 학습: 모르겠어요, 알아요, 다시듣기
- 전체 재생: 모르는 문장, 아는 문장, 일시정지

## 에빙하우스 망각곡선 레벨

| 레벨 | 복습 간격 | 설명 |
|------|----------|------|
| Lv0 | 즉시 | 방금 학습 / 틀림 |
| Lv1 | 1시간 후 | 첫 복습 |
| Lv2 | 1일 후 | 단기 기억 |
| Lv3 | 2일 후 | 기억 강화 |
| Lv4 | 4일 후 | 중기 기억 |
| Lv5 | 7일 후 | 장기 기억 시작 |
| Lv6 | 15일 후 | 장기 기억 |
| Lv7 | 30일 후 | **완전 습득** |

- 정답: 다음 레벨로 진급
- 오답: Lv0으로 리셋
- Lv7 도달 시: 복습 큐에서 제거, "mastered" 상태
